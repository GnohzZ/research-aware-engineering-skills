# Research Gate Skills

Research-aware engineering skills for routing engineering tasks through prior-art search, evidence extraction, novelty audit, experiment planning, and claim writing.

## Contents

- `skills/research-gate/` - entrypoint for deciding research intensity.
- `skills/research-project-init/` - repository initializer for persistent workitems and tooling.
- `skills/prior-art-scout/` - prior-art search workflow.
- `skills/evidence-extractor/` - evidence card extraction workflow.
- `skills/novelty-auditor/` - novelty comparison workflow.
- `skills/method-transfer/` - translate evidence into implementation guidance.
- `skills/experiment-planner/` - experiment planning workflow.
- `skills/research-ledger/` - durable research ledger maintenance.
- `skills/claim-writer/` - contribution and claim drafting.
- `skills/spike-wrap-up/` - package spike findings into durable project context.

## Install

Copy the desired skill directories into an agents skills directory, for example:

```bash
cp -a skills/* ~/.agents/skills/
```
